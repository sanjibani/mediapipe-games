# MediaPipe Games

A progressive series of browser-based games that teach MediaPipe, Canvas, Web Audio, and Three.js — from simple hand tracking to full 3D shader effects.

## Projects (Easiest → Hardest)

| # | Game | What You Learn | Status |
|---|------|---------------|--------|
| 1 | Hand Pong | MediaPipe hand tracking basics | 🔨 Building |
| 2 | Finger Fruit Slicer | Multiple finger tracking + collision | ⬜ |
| 3 | Rock Paper Scissors | Gesture classification | ⬜ |
| 4 | Head-Dodge Falling Objects | Face/body tracking + game loop | ⬜ |
| 5 | Squat Jumper | Full body pose tracking | ⬜ |
| 6 | Air Drum Kit | Web Audio API + gesture velocity | ⬜ |
| 7 | 3D Block Stacker | Three.js + hand tracking in 3D | ⬜ |
| 8 | Webcam Portal | Shaders + webcam texture | ⬜ |

## Running Any Game

Each game is a standalone HTML file. Just open `level-X-*/index.html` in a browser (or use a local server for webcam access).

```bash
# Quick local server
npx serve .
```
